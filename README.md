# Simple Calculator on WPF

A basic calculator application created to explore WPF development features. 

## Features
- **Basic Operations**:
  - Addition (+)
  - Subtraction (-)
  - Multiplication (*)
  - Division (/)
- **Error handling** for division by zero
- **Responsive design** that works on different window sizes
